دليل فك شفرات وأزرار استضافة Karo

شرح تفصيلي لكل خيار وزر موجود داخل استضافة Karo الشهيرة لتعديل جميع إصدارات PS4.

1. شريط الأدوات العلوي في الاستضافة

  • Status: يوضح لك حالة الاتصال بالذاكرة المؤقتة (Cache) وهل المتصفح مستعد لتطبيق الثغرة أم لا.
  • Copy Cheat: أداة لنسخ وتثبيت ملفات الغش والأكواد السرية (Cheats) على جهازك لاستخدامها داخل الألعاب.
  • WEB ESP / Binloader: أدوات متقدمة مخصصة لاستقبال وبث الأكواد عبر الشبكة المحلية (غالباً للمطورين أو لتشغيل أدوات خارجية).

2. ماذا تعني الحروف والرموز بجانب أرقام الإصدارات؟ (5.05, 6.72, 7.02, 7.55, 9.00)

عندما تفتح الاستضافة، ستجد خيارات متعددة لكل إصدار، وإليك معناها بالكامل:

  • الرقم الأساسي (مثل 5.05 أو 6.72): الخيار الافتراضي لتشغيل الثغرة الخام للنظام.
  • Goldhen (مثل Goldhen505 / Goldhen672 / Goldhen900): الزر الأهم والأكثر استخداماً، يقوم بتفعيل الثغرة وتفعيل قائمة الـ GoldHEN مباشرة بضغطة واحدة (وهي التي تفتح مثبت الألعاب وإعدادات التعديل).
  • الحرف L (مثل 6.72L / 7.02L / 9.00L): اختصار لـ Loader (محمل)، مخصص لتحميل أدوات برمجية مساعدة أو ملفات بين معينة.
  • كلمة Gold (مثل 7.02 Gold): نسخ معدلة ومدمجة لتحسين استقرار الثغرة وتقليل مشاكل تهنيج الجهاز (Kernel Panic).
  • الحرف A أو NM (مثل 6.72A أو 9.00NM):
    • 6.72A: نسخة معدلة بواسطة المطور (Alih) لتحسين الأداء.
    • 9.00NM: اختصار لـ (No MurtZ)، نسخة معدلة بدون شاشات ترحيب مزعجة أو مخصصة للسرعة.
  • الـ ESP (مثل ESP505tr / ESP900tr): مخصصة للأجهزة اللي بتشغل الاستضافة محلياً عبر راوتر صغير أو قطعة هاردوير خارجية (ESP8266).

3. أزرار الصيانة السفلية في الاستضافة

  • PS4Trainer Offline: تشغيل أداة الغش الشهيرة وتعديل الألعاب بدون الحاجة لاتصال إنترنت دائم.
  • Restore Host / Restore Cache: أزرار مهمة جداً لمسح وإعادة ضبط الذاكرة المؤقتة للمتصفح إذا علق الموقع أو حدثت مشكلة في التحميل لتتمكن من البدء نظيفاً.
